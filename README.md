🧠 Implementation Strategy

Each solution generally follows this workflow:

1. Complexity Analysis
   Estimate time and space complexity before coding (e.g., ensuring an O(n log n) approach when required).

2. Edge Case Handling
   Consider cases like empty inputs, extreme values, and special constraints.

3. Clean Implementation
   After passing all test cases, refine the code for readability and efficiency.
